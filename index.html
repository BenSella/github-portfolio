  • Labels (C#) project, application

    This project enables users to create custom family groups (e.g., coworkers, friends, or personal categories) and within each group, users can add dynamic labels. Each label can store various types of information, such as names, phone numbers, profile pictures, links (e.g., WhatsApp), and more. Users have complete flexibility to create any type of family group they desire, whether it's for organizing colleagues, tracking personal contacts, or even managing a shopping list by categorizing expenses under custom labels.

  • Data Annotation (Nest Js) project

    This project allows users to implement data annotations and validation rules in a NestJS application through a customizable JSON file. The system dynamically applies data validation rules to user input, ensuring that only valid data is accepted.

  • Postgresql (Nest Js) project

    This project demonstrates the integration of PostgreSQL with a NestJS application, Through the use of tables, table joins, and indexes, users can create an elaborate database structure to store and manage complex data efficiently. The project also includes features for saving data not only to SQL tables but also to Elasticsearch, enabling real-time search capabilitie, Users can perform complex queries and retrieve advanced results from Elasticsearch.
